# City Distance Calculator

A simple and interactive Python command-line tool designed to calculate and measure the distances and differences between various cities.

## 🚀 Features
- **Accurate Calculations:** Quick and precise distance outputs between selected cities.
- **Interactive CLI:** Simple and user-friendly command-line interface.
- **Easy to Use:** No complex setup required, just run and calculate.

## 🛠️ Technologies Used
- **Language:** Python 3

## 💻 How to Run the Project
Make sure you have Python installed on your system. Then, run the following command in your terminal:

```bash
python main.py
